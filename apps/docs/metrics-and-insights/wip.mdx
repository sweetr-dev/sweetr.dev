## How it works

The Work in Progress page provides a real-time view of all the team's open Pull Requests, grouped by their status into four columns:

- **Drafted** — PRs still in draft.
- **Pending Review** — Open PRs waiting for a review.
- **Changes Requested** — PRs where a reviewer requested changes.
- **Pending Merge** — Approved PRs waiting to be merged.

This is a focused view of the [Pull Requests](/platform/pull-requests) page, and can be used to get a quick overview of the team's work in progress.

## Use cases

- Run better daily meetings.
- Identify blocked PRs.
- Identify PRs with a lot of comments that might need attention.
